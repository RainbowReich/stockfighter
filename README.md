# Stockfighter

A gem for interacting with the Stockfighter.io API.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'stockfighter'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install stockfighter
